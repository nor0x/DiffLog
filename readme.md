# DiffLog AI

# CLI tool to generate release notes from git history by 
This CLI tool helps you generate release notes by analyzing your git commit history. It extracts relevant information from commit messages and formats it into a structured release note format.

## Features

# Different Audiences
- Generate release notes for different audiences (e.g., developers, end-users, social media).
- Customize the content and format of release notes based on the target audience.
- Use templates to create audience-specific release notes.
- Automatically include or exclude certain types of changes based on the audience.
- Support for multiple output formats (e.g., Markdown, HTML, plain text).
- Easy integration with CI/CD pipelines to automate release note generation.
- Ability to filter commits by tags, branches, or time periods.
- Option to include links to issues, pull requests, or external resources.
